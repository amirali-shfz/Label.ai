# Dataset Preprocessing

## To Use
Navigate to the `dataset_preprocessing` directory
### Download CSV files
`python download_csv.py`

### Filter the CSV files
`python filter_csv.py`

### Generate the test dataset
`python generate_test_dataset.py`
